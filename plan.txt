Theme data
User‑editable themes with Begin/End encoded as MMDD integers:

ts
interface ThemeEntry {
  Begin: number; // 505 => May 5
  End: number;   // 505 => May 5
  Theme: string; // "Cinco de Mayo"
}
Theme selection:
	•	Compute mmdd = month * 100 + day.
	•	For entries where Begin <= End, match if Begin <= mmdd <= End.
	•	For wrapped ranges (Begin > End), match if mmdd >= Begin || mmdd <= End.
	•	If multiple entries match, pick the narrowest range or the first by index.

4. Prompt rendering
Implement a reusable function (can live in shared module used by Workers and/or frontend):

ts
function renderPrompt(template: string, vars: Record<string, string | number>): string {
  return template.replace(/\{([a-zA-Z0-9_]+)\}/g, (_, key) => {
    const v = vars[key];
    return v !== undefined && v !== null ? String(v) : `{${key}}`;
  });
}
Before calling this, map PromptVariables to a flat object with keys matching placeholders in templates.
